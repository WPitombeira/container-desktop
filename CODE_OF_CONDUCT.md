# Contributor Covenant Code of Conduct

## 1. Purpose

We aim to build a welcoming environment for everyone who participates in Container Desktop.

We do not tolerate harassment, discrimination, or personal attacks, and expect contributors to communicate respectfully with maintainers and users.

## 2. Expected Behavior

- Be respectful and considerate in all project communication.
- Welcome newcomers and help them contribute.
- Focus criticism on code and behavior, not people.
- Keep discussions evidence-based and actionable.

## 3. Unacceptable Behavior

- Harassment, abuse, threats, or intimidation.
- Discriminatory or derogatory language and behavior.
- Repeatedly disrupting project discussions or review processes.
- Publishing private information without consent.

## 4. Scope

This code applies to all project spaces: repository issues, pull requests, and any project-linked communications.

## 5. Enforcement

Maintainers may remove, edit, or reject contributions and moderate discussions that violate this code.

## 6. Reporting

If you experience or witness a violation, open a private report on the repository and include:

- Incident context
- Participants
- Relevant links or artifacts

## 7. Attribution

This Code of Conduct is adapted from the Contributor Covenant.
